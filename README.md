# SEO Tools Frontend

A modern SEO tools platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Modern tech stack (Next.js, TypeScript, Tailwind CSS)
- Responsive design
- SEO optimization tools
- Performance analytics

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/webbuddyllc/seo-tools-frontend.git
   cd seo-tools-frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Copy the environment variables:
   ```bash
   cp .env.example .env.local
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your values:

- `NEXT_PUBLIC_API_URL` - Backend API URL
- `NEXT_PUBLIC_AUTH_DOMAIN` - Authentication domain
- `NEXT_PUBLIC_AUTH_CLIENT_ID` - Authentication client ID
